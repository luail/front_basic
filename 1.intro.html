HTML 학습하기

H태그는 제목을 의미하고 위아래로 약간의 여백이 자동삽입되고 줄바꿈.

검색엔진에서 h태그는 중요한 검색 키워드로 사용

p태그는 단락구분 태그이고, 줄바꿈이 되면서 태그 위아래로 약간의 여백이 삽입

html 태그 내에서는 엔터를 치더라도 줄바꿈이 되지 않고,
<br>태그를 써야 줄바꿈이 됨에 유의. 또한 웹브라우저에서는 여러번 띄어쓰기를 하더라도 한번만 띄어쓰기가 됨에 유의.

텍스트의 형식

굵게와 강조된 굵게가 사용되고, 이텔릭체와 강조된 이텔릭체가 있다. 삭제된 텍스트를 표현할때 del태그 사용
여러개도 같이 사용된당

html에서 엔티티(특수기호) 사용

단락을 표시하는 태그. <p> 태그이다.

여러개의 공백을 넣고 싶을 때에는      nbsp를 사용한다.

div태그는 줄바꿈되는 단락처럼 사용되나, 별다른 기능이 없고 다른 HTML요소들을 하나로 묶어 디자인을 적용하는데 주로 사용.
span 태그는 별다른 기능이 없고, html요소들에 css 적용을 위해 주로 사용. 줄바꿈이 없다.
